# 第4章 動的計画法

## 4.0 動的計画法とは
- 動的計画法(Dynamic Programming: DP)
    - より小さい問題の結果を利用して問題を解く方法

## 4.1 動的計画法の基本

### A16 Dungeon1
dp[i] := 部屋1から部屋iまでの最短時間

部屋iへの行き方は部屋i-1からA[i]の道を通る
もしくは，部屋i-2からB[i]の道を通る

よって，
```
dp[i] = min(dp[i-1]+A[i], dp[i-2]+B[i])
```
